## basic

### create `.env` file following:

```env
NODE_ID=
NODE_ROLE=manager
NETWORK_MODE=overlay

VERSION=latest

DOMAIN=hive.home.local
PROTOCOL=http
PORT=9000

MAX_HEAP_SIZE=1G
HEAP_NEWSIZE=1G
```

## info

default login credentials are:

- login: `admin@thehive.local`
- password: `secret`
